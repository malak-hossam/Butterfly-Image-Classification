# Data Card

## Dataset

- Name: Butterfly Images 40 Species (Kaggle)
- Source: https://www.kaggle.com/datasets/gpiosenka/butterfly-images40-species
- Task: Fine-grained image classification of butterfly/moth species
- Format: Folder-based image dataset compatible with `torchvision.datasets.ImageFolder`

## Expected Local Layout

```text
data/raw/butterfly-images40-species/
  train/<class_name>/*.jpg
  valid/<class_name>/*.jpg
  test/<class_name>/*.jpg
```

## Label Space

- Number of classes inferred from `train/` folder at runtime.
- `class_to_idx` mapping is generated by ImageFolder and logged in sanity checks.

Notebook reference example reported 100 classes with class names such as `ROSY MAPLE MOTH`, `YELLOW SWALLOW TAIL`, etc.

## Split Details

Notebook reference output:

- Train: 12,594 images
- Valid: 500 images
- Test: 500 images

These counts should be validated locally with `python -m src.data.sanity_check`.

## Data Quality and Limitations

- Potential class imbalance across species.
- Visual similarity across related species can increase confusion.
- Image conditions vary (background, lighting, angle), which helps robustness but also adds noise.
- Dataset provenance and labeling process are external to this repository.

## Responsible Use

- Intended for research/education and model engineering practice.
- Not intended for high-stakes biodiversity decisions without external verification.

## Privacy and Licensing

- Respect Kaggle dataset terms and licensing.
- This repository intentionally excludes dataset image files from version control.
